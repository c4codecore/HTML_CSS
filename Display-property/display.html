CSS Display Property — Visual Learning

Har example mein pehle DEFAULT behavior dikhaya hai, phir display property lagane ke baad kya badlav aata hai — tabhi samajh aata hai!

1. display: block

Block element puri width leta hai aur next element neeche aa jaata hai.
Kya seekhna hai: <span> by default inline hota hai. Isko display: block lagao — toh wo block jaisa behave karega.

❌ PEHLE — span ka default behavior (inline)

Default: display inline
Span 1 Span 2 Span 3

👆 Teeno span ek hi line mein hain — kyunki span inline hota hai by default.

✅ BAAD MEIN — span ko block banaya

display: block lagaya
Span 1 — ab block hai Span 2 — ab block hai Span 3 — ab block hai

👆 Ab har span apni puri row le raha hai, width bhi set ho gayi — display: block ka effect!

span {
    display: block;   /* inline span ko block bana diya */
    width: 280px;     /* ab width kaam karegi */
}

2. display: inline

Inline element sirf utni hi jagah leta hai jitni zaroorat ho. Width/height set nahi hoti.
Kya seekhna hai: <div> by default block hota hai. Isko display: inline lagao — toh wo side-by-side aa jayega.

❌ PEHLE — div ka default behavior (block)

Default: display block

👆 Teeno div ek ke neeche hain — kyunki div block hota hai by default.

✅ BAAD MEIN — div ko inline banaya

👆 Ab teeno div side-by-side hain — display: inline ka effect!

⚠️ Dhyan do: inline pe width / height kaam nahi karta.

div {
    display: inline;   /* block div ko inline bana diya */
    /* width: 200px;  ← yeh kaam NAHI karega */
}

3. display: inline-block

inline-block = inline + block ka combination. Element inline flow mein rehta hai aur width/height bhi set hoti hai.
Kya seekhna hai: <span> inline hota hai — isme width/height set nahi hoti. inline-block lagane se dono kaam hoga.

❌ PEHLE — span par width/height ka koi asar nahi

Default: display inline — width/height ignored
Box A (120×60 set ki, par kaam nahi ki) Box B Box C

👆 Width 120px aur height 60px set ki thi — lekin inline hone ki wajah se apply nahi hui.

✅ BAAD MEIN — inline-block se width/height bhi kaam ki

display: inline-block lagaya
Box A Box B Box C

👆 Ab teeno ek line mein hain aur equal size ke hain — inline-block ka dual fayda!

Real use case

Navigation buttons, image galleries, cards — jab items ek line mein bhi chahiye aur size bhi control karna ho.

span {
    display: inline-block;   /* inline + block ka fayda dono */
    width: 120px;            /* ab kaam karega ✅ */
    height: 60px;            /* ab kaam karega ✅ */
}

4. display: none — aur visibility: hidden se farq

Dono element ko hide karte hain, lekin ek important farq hai:

  • display: none — element bilkul gayab — layout mein jagah bhi nahi lega
  • visibility: hidden — element dikhta nahi, lekin jagah reserve rehti hai

display: none ka example

Box 2 display:none hai
Box 1 ✅
Box 2 — display:none (gayab!)
Box 3 ✅

👆 Box 2 poori tarah gayab hai — Box 1 aur Box 3 directly adjacent hain.

visibility: hidden ka example

Box 2 visibility:hidden hai
Box 1 ✅
Box 2 — hidden (jagah hai)
Box 3 ✅

👆 Box 2 dikhta nahi, lekin uski jagah bachi hui hai — isliye Box 3 door hai.

/* Poora hata do — jagah bhi nahi lega */
.gone { display: none; }

/* Sirf chhupao — jagah reserve rahegi */
.invisible { visibility: hidden; }

5. display: flex

flex lagane se container ke andar ke elements automatically horizontal row mein aa jaate hain aur easily align kiye ja sakte hain.
Kya seekhna hai: Bina flex ke divs stack hote hain. Flex lagao — alignment magic shuru!

❌ PEHLE — divs stack ho rahe hain (block default)

No flex — default block stacking

✅ BAAD MEIN — display:flex lagaya

👆 Teeno ek row mein aur center aligned — sirf ek line se!

.container {
    display: flex;        /* items horizontal row mein */
    gap: 10px;            /* items ke beech gap */
    align-items: center;  /* vertically center */
}

6. display: grid

grid se 2D layout ban sakta hai — rows aur columns dono control kar sakte hain.
Kya seekhna hai: Bina grid ke divs ek column mein stack hote hain. Grid lagao — multi-column layout!

❌ PEHLE — divs ek column mein (block default)

👆 6 cards 3-column grid mein — sirf CSS se!

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
    gap: 10px;
}

7. display: inline-flex (Bonus)

inline-flex = flex container jo inline flow mein rahega. Real use case: Navigation bar jo text ke saath inline ho.

Real use case — Nav bar

Page header mein yeh nav hai:

  • Home
  • About
  • Contact
— aur text saath chal sakta hai. 

.nav {
    display: inline-flex;   /* flex + inline dono */
    gap: 6px;
}

📋 Quick Summary

Value Puri Width leta hai? Width/Height set hoti hai? Side-by-side hota hai? Common use
block ✅ Haan ✅ Haan ❌ Nahi div, p, h1, section
inline ❌ Nahi ❌ Nahi ✅ Haan span, a, strong
inline-block ❌ Nahi ✅ Haan ✅ Haan Buttons, nav items
none Element poori tarah gayab — layout mein jagah nahi Hide karna
flex Children horizontal row mein, alignment easy Navbar, cards row
grid Children rows + columns dono mein Page layout, gallery
