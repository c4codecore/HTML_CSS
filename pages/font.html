🔤 Fonts & Text Styling

CSS mein text ko style karne ki saari important properties — har ek ka live example neeche hai.

1. font-family

font-family se choose karte hain ki text kis font mein dikhe. Hamesha 2-3 fonts likhein — agar pehla nahi mila to dusra use hoga.
Format: font-family: PrimaryFont, FallbackFont, generic-family;

Arial, sans-serif — Yeh ek sans-serif font hai (letters ke upar/neeche chhoti lines nahi hoti)
"Times New Roman", serif — Yeh ek serif font hai (letters par decorative lines hoti hain)
"Courier New", monospace — Yeh monospace hai — har letter same width ka hota hai (code likhne mein use)
Georgia, serif — Serif font, reading ke liye comfortable 
/* font-family: PrimaryFont, FallbackFont, generic; */

.arial-text   { font-family: Arial, sans-serif; }
.times-text   { font-family: "Times New Roman", serif; }
.courier-text { font-family: "Courier New", monospace; }
.georgia-text { font-family: Georgia, serif; }

2. font-size

font-size text ka size set karta hai. px fixed hota hai, em parent ke relative, rem root ke relative.

font-size: 12px — Chhota text (captions, footnotes)
font-size: 16px — Normal text (browser ka default)
font-size: 24px — Bada text (subheadings)
font-size: 2em — Parent ka 2 guna (yahan = 32px)
font-size: 1.5rem — Root ka 1.5 guna (= 24px)
/* px — fixed size */
.small  { font-size: 12px; }
.normal { font-size: 16px; }   /* browser default */
.large  { font-size: 24px; }

/* em — parent ke font-size ke relative */
.em     { font-size: 2em; }    /* parent = 16px, to 2em = 32px */

/* rem — html tag ke font-size ke relative (zyada predictable) */
.rem    { font-size: 1.5rem; } /* html = 16px, to 1.5rem = 24px */

3. font-weight

font-weight se text kitna mota ya patla dikhega. 100 se 900 tak values hain. 400 = normal, 700 = bold.

font-weight: 100 — Thin (bahut patla)
font-weight: 400 — Normal (default)
font-weight: 700 — Bold (mota)
font-weight: 900 — Extra Bold (bahut mota)
/* Number values use kar sakte hain */
.thin        { font-weight: 100; }
.normal      { font-weight: 400; }   /* default */
.bold        { font-weight: 700; }
.extra-bold  { font-weight: 900; }

/* Ya words bhi chal jaate hain */
.normal-word { font-weight: normal; }  /* = 400 */
.bold-word   { font-weight: bold; }    /* = 700 */

4. font-style

font-style: italic text ko teda (slanted) kar deta hai. Mostly quotes, terms, ya emphasis ke liye use hota hai.

font-style: normal — Seedha text (default)
font-style: italic — Teda text (italicized)
.normal { font-style: normal; }  /* default */
.italic { font-style: italic; }  /* teda kar do */

5. line-height

line-height lines ke beech vertical space set karta hai. Unit-less number likhna best hai (e.g. 1.5) — font-size ke relative hota hai.

line-height: 1.0 — Cramped text
Lines bahut paas hain.
Padhna mushkil lagta hai.
line-height: 1.5 — Comfortable reading
Lines ke beech theek jagah hai.
Padhne mein aasaan lagta hai.
line-height: 2.2 — Very spacious
Lines bahut door hain.
Kabhi kabhi poetry mein use hota hai. 
/* Best practice: unit-less number likhein */
.tight  { line-height: 1.0; }   /* crowded */
.normal { line-height: 1.5; }   /* comfortable — recommended */
.loose  { line-height: 2.2; }   /* very spacious */

/* Pixel mein bhi de sakte hain */
.px     { line-height: 24px; }

6. text-align

text-align text ko container ke andar kahan rakhna hai ye batata hai.

text-align: left — Left se shuru (default)
text-align: center — Beech mein
text-align: right — Right side
text-align: justify — Dono taraf equal margin. Newspaper mein aisa hota hai. Jab text bahut lamba ho tab yeh useful hota hai kyunki dono edges bilkul seedhi lagti hain. 
.left    { text-align: left; }     /* default */
.center  { text-align: center; }
.right   { text-align: right; }
.justify { text-align: justify; }  /* dono side equal — newspaper style */

7. text-decoration

text-decoration text par line lagata ya hatata hai. Links pe by default underline hoti hai — none se hata sakte hain.

text-decoration: underline — Neeche line
text-decoration: overline — Upar line
text-decoration: line-through — Beech se line (strikethrough)
text-decoration: none — Koi line nahi (links ki default underline hatane ke liye use karo) 
.underline    { text-decoration: underline; }    /* neeche line */
.overline     { text-decoration: overline; }     /* upar line */
.linethrough  { text-decoration: line-through; } /* beech se kata */
.none         { text-decoration: none; }         /* link ki underline hatao */

8. text-transform

text-transform HTML mein jo bhi likha ho, uska case CSS se badal sakte hain. Useful hai jab heading hamesha uppercase chaahiye — HTML mein normal likh sakte hain.

text-transform: uppercase — YEH TEXT SAB CAPITAL MEIN DIKHEGA
text-transform: lowercase — YEH TEXT SAB CHHOTA MEIN DIKHEGA
text-transform: capitalize — Har Word Ka Pehla Letter Capital
.upper      { text-transform: uppercase; }   /* SABS CAPS */
.lower      { text-transform: lowercase; }   /* sab chhota */
.capitalize { text-transform: capitalize; }  /* Har Word Capital */

9. letter-spacing & word-spacing

letter-spacing — har letter ke beech ki jagah. word-spacing — har word ke beech ki jagah.

letter-spacing: -1px — Letters paas paas (tight)
letter-spacing: normal — Default spacing
letter-spacing: 4px — Letters door door (wide) — HEADINGS MEIN ACCHA LAGTA HAI
word-spacing: normal — Words ke beech normal gap
word-spacing: 12px — Words ke beech zyada gap
/* Letter spacing */
.tight  { letter-spacing: -1px; }   /* letters paas */
.normal { letter-spacing: normal; } /* default */
.wide   { letter-spacing: 4px; }    /* letters door */

/* Word spacing */
.ws-normal { word-spacing: normal; }
.ws-wide   { word-spacing: 12px; }

📋 Quick Reference — Saari Properties

Property Kya karta hai Common Values
font-family Font choose karo Arial, "Times New Roman", monospace
font-size Text ka size 16px, 1.5rem, 2em
font-weight Text kitna mota/patla 100–900, normal, bold
font-style Text teda karo normal, italic
line-height Lines ke beech ka space 1.5, 2, 24px
text-align Text left/center/right left, center, right, justify
text-decoration Underline, overline, strikethrough underline, overline, line-through, none
text-transform Case change karo uppercase, lowercase, capitalize
letter-spacing Letters ke beech space normal, 2px, -1px
word-spacing Words ke beech space normal, 8px
