📦 CSS Box Model

CSS mein har element ek rectangular box hoti hai — Content → Padding → Border → Margin. Yeh chaar layers milke ek element banate hain.

Box Model — Visual Diagram

Har element mein 4 layers hoti hain. Bahar se andar jaate hain to: Margin → Border → Padding → Content

MARGIN (transparent — page background dikhta hai)
BORDER
PADDING (background color dikhti hai)
CONTENT
(text / image yahan hota hai) 
/* Box model ka formula (default) */
Total Width  = width + padding-left + padding-right + border-left + border-right
Total Height = height + padding-top + padding-bottom + border-top + border-bottom

1. Content — Actual text ya image

width aur height se content area ka size set karte hain.

Yeh content area hai — width: 300px, height: 80px set ki hai 
div {
    width: 300px;       /* content ka width */
    height: 80px;       /* content ka height */
    background: lightblue;
}

2. Padding — Content aur border ke beech ka space

Padding andar ka space hai. Background color padding mein bhi dikhti hai. Padding se content "breathe" karta hai — text border se chipka nahi lagta.

Shorthand Rule → charon values ghadi ki direction mein: Top → Right → Bottom → Left
padding: 20px — Charon taraf 20px
padding: 10px 30px — Top+Bottom=10px, Left+Right=30px
padding: 10px 20px 30px — Top=10, L+R=20, Bottom=30
padding: 10px 20px 30px 40px — Top=10, Right=20, Bottom=30, Left=40
Individual sides
padding-top:20 | padding-right:40 | padding-bottom:10 | padding-left:60
/* Shorthand — ek value */
div { padding: 20px; }                /* charon taraf 20px */

/* Shorthand — do values */
div { padding: 10px 30px; }           /* top+bottom, left+right */

/* Shorthand — teen values */
div { padding: 10px 20px 30px; }      /* top, left+right, bottom */

/* Shorthand — char values (Top Right Bottom Left — TRBL) */
div { padding: 10px 20px 30px 40px; }

/* Individual sides */
div {
    padding-top:    20px;
    padding-right:  40px;
    padding-bottom: 10px;
    padding-left:   60px;
}

3. Border — Element ke charon taraf line

Border ek visible line hai jo element ke around lagti hai. Shorthand: border: width style color; — style likhna zaroori hai (solid, dashed, dotted, double).

Border Styles
border: 4px solid red — solid line
border: 4px dashed blue — dashed line
border: 4px dotted green — dotted line
border: 8px double purple — double line
Individual sides — har taraf alag border
Top=solid-red | Right=dashed-blue | Bottom=dotted-green | Left=double-orange
border-radius — corners ko round karo
border-radius: 4px — thoda round
border-radius: 12px — medium round
border-radius: 24px — zyada round (pill shape)
border-radius: 50% = circle
/* Shorthand */
div { border: 4px solid red; }     /* width style color — teeno zaroori */

/* Alag alag styles */
div { border: 4px dashed blue; }
div { border: 4px dotted green; }
div { border: 8px double purple; }

/* Individual sides */
div {
    border-top:    4px solid red;
    border-right:  4px dashed blue;
    border-bottom: 4px dotted green;
    border-left:   4px double orange;
}

/* Border radius */
div { border-radius: 4px; }    /* thoda round */
div { border-radius: 50%; }    /* circle */

4. Margin — Element ke bahar ka space

Margin element ke bahar hoti hai — dusre elements se door karne ke liye. Margin transparent hoti hai — page background dikhti hai.
Special trick: margin: auto se element center ho jaata hai!

margin: 20px — Charon taraf 20px space
margin: 10px 40px — Top+Bottom=10, Left+Right=40
margin: 10px 20px 30px 40px — Top Right Bottom Left
margin: auto — Element ko center karna
margin: 20px auto; — Yeh box center mein hai!
(Width set karna zaroori hai) 
/* Shorthand — padding jaisa hi */
div { margin: 20px; }                /* charon taraf */
div { margin: 10px 40px; }           /* top+bottom, left+right */
div { margin: 10px 20px 30px 40px; } /* top right bottom left */

/* Center karna — left+right auto */
div {
    width: 300px;        /* width zaroori hai */
    margin: 0 auto;      /* left+right auto = center mein */
}

/* Individual sides */
div {
    margin-top:    20px;
    margin-right:  40px;
    margin-bottom: 10px;
    margin-left:   60px;
}

5. box-sizing — SABSE IMPORTANT concept

Yeh bahut confusing hota hai beginners ke liye!
Default mein padding aur border width ke UPAR se add hoti hai — actual size badi ho jaati hai. border-box se yeh problem solve hoti hai.

box-sizing: content-box (DEFAULT — confusing!)
width: 200px set ki — lekin actual width 250px render hogi!
(200 content + 40 padding + 10 border = 250px)
box-sizing: border-box (RECOMMENDED — predictable)
width: 200px set ki — actual width exactly 200px rahegi!
(padding + border andar fit hote hain) 
/* DEFAULT — content-box (confusing) */
.box {
    box-sizing: content-box;   /* default — mention karna zaroori nahi */
    width: 200px;
    padding: 20px;
    border: 5px solid red;
    /* ACTUAL WIDTH = 200 + 40 + 10 = 250px  ← unexpected! */
}

/* RECOMMENDED — border-box (predictable) */
.box {
    box-sizing: border-box;    /* width = total width */
    width: 200px;
    padding: 20px;
    border: 5px solid green;
    /* ACTUAL WIDTH = 200px exactly  ← expected! */
}

/* Best practice — sabhi elements par laga do */
* {
    box-sizing: border-box;
}

📋 Summary — Box Model

Padding Content aur border ke beech andar ka space ✅ Haan padding
Border Element ke around line / frame Border khud visible hoti hai border
Margin Element ke bahar ka space ❌ Transparent margin
