🎭 Pseudo-Classes & Pseudo-Elements

Pseudo-class (:) = element ki ek STATE ya POSITION target karo.
Pseudo-element (::) = element ke ek KHAAS HISSE ko style karo ya content add karo.

PSEUDO-CLASSES — Element ki State target karo

Pseudo-classes kisi element ki current state ya position ke hisaab se style karte hain. Single colon use hota hai: element:state

1. :hover — Mouse element par aaye tab

Jab user mouse pointer element ke upar laaye tab style apply hoti hai.
Buttons, links, cards — sabme hover use hota hai interactivity ke liye.

Link Button — Hover karo 🖱️ Doosra Button 
/* Normal state */
.btn {
    background: #3b82f6;
    color: white;
    padding: 10px 24px;
}

/* Mouse aane par */
.btn:hover {
    background: #1d4ed8;         /* rung badal gaya */
    transform: translateY(-2px); /* thoda upar uthega */
}

2. :active — Click karte waqt (mouse button dabane par)

Jab element par mouse button dabaya jaaye tab apply hota hai. Button press hone ka physical feel deta hai. 

.btn:active {
    background: #1e40af;       /* click ke waqt aur dark */
    transform: scale(0.97);    /* thoda chhota = pressed feel */
}

3. :focus — Element focus mein ho tab

Jab element active/selected ho — input click karo ya Tab key se navigate karo.
Accessibility ke liye bahut zaroori hai — keyboard users ke liye clearly dikhna chahiye.
Default browser outline hatao to khud custom style do! 

input {
    border: 2px solid #cbd5e1;   /* normal border */
    outline: none;               /* default browser outline hataya */
}

input:focus {
    border-color: #3b82f6;       /* focus par blue border */
    background: #eff6ff;         /* light blue background */
    /* outline: none already set hai — apna style diya */
}

4. :visited — Visit ki ja chuki link

Jab user kisi link par pehle ja chuka ho to alag style. Default mein visited links purple hoti hain — aap customize kar sakte hain.

Google (ek baar visit karo — phir color badlega)

Example.com 
a           { color: #3b82f6; }   /* unvisited — blue */
a:visited   { color: #8b5cf6; }   /* visited — purple */

5. :checked — Checkbox ya radio selected ho

Jab checkbox ya radio button select ki jaaye tab style apply hoti hai. + selector se adjacent sibling (label) ko bhi style kar sakte hain. 

/* Checkbox checked hone par uske baad wale label ko style karo */
input[type="checkbox"]:checked + label {
    color: green;
    font-weight: bold;
}

/* + matlab adjacent sibling (turant baad wala element) */

6. :disabled — Disabled input

disabled attribute wale form elements ko alag style dena. User clearly samjhe ki yeh field fill nahi kar sakta. 



input:disabled {
    background: #f1f5f9;      /* grey background */
    color: #94a3b8;           /* light text */
    cursor: not-allowed;      /* no-entry cursor */
    border-color: #e2e8f0;
}

7. :first-child, :last-child, :nth-child() — Position ke hisaab se

Kisi parent ke andar position ke hisaab se elements target karo.
:first-child → pehla, :last-child → aakhri, :nth-child(n) → nth number wala (1 se start hota hai).
:nth-child(even) → even numbers, :nth-child(odd) → odd numbers.

List — pehla red, doosra highlighted, aakhri blue
  • Pehla item — :first-child (red bold)
  • Doosra item — :nth-child(2) (yellow background)
  • Teesra item — normal
  • Chautha item — normal
  • Paanchwa item — :last-child (blue bold)

:not(selector) se ek specific element ko style se bahar rakh sakte hain.

PSEUDO-ELEMENTS — Element ke hisse ko style karo

Pseudo-elements element ke ek specific part ko target karte hain, ya fake content add karte hain. Double colon use hota hai: element::pseudo-element

9. ::before — Content ke PEHLE content add karo

Element ke content se pehle ek virtual element add karta hai. content: "" property likhna zaroori hai (chahe khaali ho).
HTML change kiye bina decorative content add kar sakte hain.

Yeh paragraph hai — pehle emoji add ki CSS se

Doosra paragraph — HTML same hai, CSS ne emoji add ki

.item::before {
    content: "🔷 ";   /* yeh HTML mein nahi hai — CSS ne add kiya */
    color: #3b82f6;
}

/* Content values:
   content: "text"     → plain text
   content: "🔷"       → emoji
   content: ""         → khaali (decoration ke liye)
   content: attr(data-x) → HTML attribute ki value */

10. ::after — Content ke BAAD content add karo

Element ke content ke baad ek virtual element add karta hai. Decorative elements, badges, labels ke liye use hota hai.

Yeh text hai — baad mein checkmark add hua CSS se

Doosra paragraph bhi check ho gaya

.item::after {
    content: " ✅";   /* element ke baad add hoga */
    color: green;
}

11. ::before / ::after — Decorative use (underline effect)

Sirf decoration ke liye content: "" (khaali) likhte hain. Decorative line, badge, ribbon — sab bina extra HTML ke banana possible hai.

Heading with Underline via ::after

.heading::after {
    content: "";               /* khaali — sirf decoration */
    display: block;
    width: 60px;
    height: 3px;
    background: #3b82f6;       /* blue line */
    margin: 8px auto 0;
    border-radius: 2px;
}

/* Content khaali rakha — sirf ek coloured line chahiye thi */

12. ::first-letter — Pehla letter alag style

Paragraph ka sirf pehla letter style karo — newspaper ya book jaisa effect. Drop cap effect kehte hain isse.

Yeh ek paragraph hai. Pehla letter bada aur red hai — ::first-letter se. Baaki text normal size mein hai. Yeh newspaper mein common hota hai. 

p::first-letter {
    font-size: 48px;     /* bada letter */
    font-weight: bold;
    color: #dc2626;      /* red color */
    float: left;         /* text ke saath wrap karne ke liye */
    margin-right: 4px;
    line-height: 1;
}

13. ::first-line — Pehli line alag style

Paragraph ki sirf pehli visible line ko style karo. Screen ka size badlega to bhi pehli line automatically re-calculate hogi.

Yeh pehli line hai — blue aur bold hai ::first-line se.
Yeh doosri line hai — normal style mein.
Teesri line bhi normal hai. 

p::first-line {
    color: #2563eb;      /* pehli line blue */
    font-weight: bold;
    font-size: 17px;
}

14. ::selection — Text select karne par

User jab text select kare (mouse drag kare) to selection ka color customize kar sakte hain. Default browser selection blue hoti hai.
👇 Neeche ke text ko select karke dekho!

Is text ko mouse se select karo — selection ka color yellow ho jaayega! Normal text selection se alag experience milega. 

.text::selection {
    background: #fbbf24;   /* selection ka background — yellow */
    color: #1c1917;        /* selected text ka color — dark */
}

15. ::placeholder — Input placeholder style

Input ya textarea ke placeholder text ko style karo. Default grey boring hota hai — brand ke hisaab se customize kar sakte hain. 

input::placeholder {
    color: #94a3b8;       /* placeholder ka color */
    font-style: italic;   /* italic */
}
