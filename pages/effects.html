✨ CSS Visual Effects

Transition, Transform, Animation, Gradient, Shadow, Opacity, Overflow, Backdrop-filter, Gradient Text — yeh sab milke ek page ko professional aur attractive banate hain.

1. transition — Smooth change

Jab koi CSS property change hoti hai (jaise hover pe) to by default woh instantly badal jaati hai. transition se woh change smoothly hota hai.
⚡ Rule: transition hamesha us element par likhte hain jo change hoga — :hover par nahi!

❌ Bina transition — instantly change hota hai
✅ Transition ke saath — smoothly change hota hai 
/* ❌ Bina transition */
.btn { background-color: blue; }
.btn:hover { background-color: red; }   /* instantly red */

/* ✅ Transition ke saath */
.btn {
    background-color: blue;
    transition: background-color 0.4s ease;  /* yahan likhte hain */
}
.btn:hover { background-color: red; }   /* smoothly red */
Multiple properties — alag alag time se 
/* Multiple properties */
.btn {
    transition: background-color 0.3s ease,
                border-color 0.3s ease,
                font-size 0.3s ease;
}

/* Ya sab ek saath */
.btn { transition: all 0.3s ease; }
transition: all — shape bhi smoothly badlegi
Timing functions — speed ka pattern (sab par hover karo, farq dekho)
ease (default)
linear
ease-in
ease-out
transition: background-color 1s ease;       /* slow-fast-slow (default) */
transition: background-color 1s linear;     /* same speed throughout */
transition: background-color 1s ease-in;    /* slow start, fast end */
transition: background-color 1s ease-out;   /* fast start, slow end */
transition-delay — thodi der baad shuru ho 
/* property  duration  timing  delay */
transition: background-color 0.4s ease 0.5s;
/*                                     ↑ 0.5 second baad shuru hoga */

2. transform — Move, Rotate, Scale, Skew

transform se element ko visually move, ghoomna, bada/chhota ya teda kar sakte hain — bina layout affect kiye. Doosre elements apni jagah nahi chhodeenge.
Transform akela use karo ya transition ke saath smooth effect ke liye.

Sab boxes par hover karo — transform dekhne ke liye

/* translateX — horizontal move */
.box:hover { transform: translateX(40px); }    /* 40px right */
.box:hover { transform: translateX(-40px); }   /* 40px left */

/* translateY — vertical move */
.box:hover { transform: translateY(-20px); }   /* 20px upar */
.box:hover { transform: translateY(20px); }    /* 20px neeche */

/* rotate — ghoomna */
.box:hover { transform: rotate(45deg); }       /* 45 degree */
.box:hover { transform: rotate(360deg); }      /* poora chakkar */
.box:hover { transform: rotate(-90deg); }      /* ulta */

/* scale — size badlana */
.box:hover { transform: scale(1.3); }          /* 1.3x bada */
.box:hover { transform: scale(0.7); }          /* 0.7x chhota */

/* skewX — teda karna */
.box:hover { transform: skewX(20deg); }

/* Multiple transforms — ek saath */
.box:hover { transform: translateY(-8px) scale(1.1); }
/* ↑ upar bhi gaya aur bada bhi hua — dono ek line mein */
✅ Sabse common use — card hover effect
🎯 Card hover karo
translateY(-8px) 
/* Card hover — translateY aur transition saath mein */
.card {
    transition: transform 0.3s ease;     /* smooth */
}
.card:hover {
    transform: translateY(-8px);         /* 8px upar utha */
}

3. @keyframes + animation — Automatic movement

transition sirf hover pe kaam karta hai.
animation automatic chalta hai — page load par, baar baar, ya kisi bhi waqt.
2 steps: (1) @keyframes mein animation ke steps likhte hain, (2) animation property se element par apply karte hain.

fadeIn — page load par appear hona
Fade In — page refresh karo
/* Step 1 — @keyframes mein steps likhte hain */
@keyframes fadeIn {
    from { opacity: 0; }   /* shuru mein invisible */
    to   { opacity: 1; }   /* end mein visible */
}

/* Step 2 — element par apply karo */
.box {
    animation: fadeIn 2s ease;
/*             ↑name ↑time ↑timing */
}
slideFromLeft — left se aana
← Slide from Left
slideUp — neeche se upar aana (delay ke saath)
↑ Slide Up (0.3s delay)
/* Slide from left */
@keyframes slideFromLeft {
    from { opacity: 0; margin-left: -100px; }
    to   { opacity: 1; margin-left: 0; }
}
.box { animation: slideFromLeft 1s ease; }

/* Slide up */
@keyframes slideUp {
    from { opacity: 0; margin-top: 40px; }
    to   { opacity: 1; margin-top: 0; }
}
/* animation-delay — 0.3s baad shuru, both → start state mein raho */
.box { animation: slideUp 1s ease 0.3s both; }
Infinite animations — baar baar chalte rehna
Pulse
Bounce
|
← Alternate →
/* Pulse — color badlata rahega */
@keyframes pulse {
    0%   { background-color: red; }
    50%  { background-color: orange; }
    100% { background-color: red; }
}
.box { animation: pulse 2s ease infinite; }

/* Bounce — upar neeche */
@keyframes bounce {
    0%   { margin-top: 0; }
    50%  { margin-top: -20px; }
    100% { margin-top: 0; }
}
.box { animation: bounce 1s ease infinite; }

/* Spinner — loading circle */
@keyframes spin {
    from { transform: rotate(0deg); }
    to   { transform: rotate(360deg); }
}
.spinner { animation: spin 1s linear infinite; }

/* Alternate — aage phir peeche */
@keyframes moveRight {
    from { margin-left: 0; }
    to   { margin-left: 200px; }
}
.box { animation: moveRight 1.5s ease infinite alternate; }

4. linear-gradient — Color blend

Do ya zyada colors ko smoothly blend karna. Yeh background-image ki jagah set hota hai — background-color nahi.
Syntax: linear-gradient(direction, color1, color2)

to right — Blue se Purple
to bottom — Green se Dark
to right bottom — Diagonal
45deg — Purple se Pink
3 colors — Red → Orange → Green
Color stops — Blue 0% → Purple 50% → Pink 100%
Sharp line — Blue 50% | Red 50% (koi blend nahi)
/* Left se right */
background: linear-gradient(to right, blue, purple);

/* Upar se neeche */
background: linear-gradient(to bottom, green, black);

/* Diagonal */
background: linear-gradient(to right bottom, orange, red);

/* Degree angle */
background: linear-gradient(45deg, purple, pink);

/* 3 colors */
background: linear-gradient(to right, red, orange, green);

/* Color stops — kahan se kahan tak */
background: linear-gradient(to right,
    blue   0%,     /* 0% tak blue */
    purple 50%,    /* 50% par purple */
    pink   100%);  /* 100% par pink */

/* Sharp line — gradient nahi, seedha cut */
background: linear-gradient(to right,
    blue 50%,      /* 50% tak blue */
    red  50%);     /* 50% se red — koi blend nahi */

/* Semi-transparent gradient — navbar mein */
background: linear-gradient(to right,
    rgba(15, 23, 42, 0.95),
    rgba(30, 41, 59, 0.95));

5. box-shadow — Element ke around shadow

Element ke bahar ya andar shadow dena.
Syntax: box-shadow: x y blur spread color;
x = horizontal, y = vertical, blur = kitni blurry, spread = kitni badi, color = shadow ka rang.

box-shadow: 4px 4px 10px rgba(0,0,0,0.2) — Simple shadow
box-shadow: 0 4px 20px rgba(0,0,0,0.1) — Soft shadow (cards mein common)
box-shadow: 0 10px 40px rgba(0,0,0,0.3) — Strong shadow
box-shadow: 0 8px 24px rgba(59,130,246,0.5) — Colored shadow (blue)
box-shadow: multiple shadows — 3D effect
box-shadow: inset — andar ki taraf (pressed look)
box-shadow: 5px 5px 0px black — Sharp (no blur) — retro style
Hover karo — shadow badhegi
/* Syntax: x  y  blur  spread  color */
box-shadow: 4px 4px 10px 0px rgba(0,0,0,0.2);
/*          ↑x  ↑y  ↑blur ↑spread ↑color    */

/* Neeche shadow (cards) */
box-shadow: 0 4px 20px rgba(0, 0, 0, 0.1);

/* Colored shadow — button ke color se match */
box-shadow: 0 8px 24px rgba(59, 130, 246, 0.5);

/* Multiple shadows */
box-shadow: 2px 2px 4px rgba(0,0,0,0.1),
           -2px -2px 4px rgba(255,255,255,0.8);

/* Inset — andar ki taraf */
box-shadow: inset 0 2px 8px rgba(0, 0, 0, 0.2);

/* Sharp — blur 0 */
box-shadow: 5px 5px 0px #000000;

opacity: 0 = poora invisible, opacity: 1 = poora visible.
⚠️ Dhyan do: opacity se poora element transparent hota hai — text, background, border sab. Agar sirf background transparent chahiye to rgba() use karo.

/* Hover pe dim karna */
.box { opacity: 1; transition: opacity 0.3s ease; }
.box:hover { opacity: 0.7; }
opacity vs rgba — ka farq

opacity: 0.5 — text bhi fade

Text + Background dono fade

rgba background — sirf background fade

Text clear, background transparent

.box {
    background-color: blue;
    opacity: 0.5;           /* text bhi 50% fade ho jaata hai */
}

.box {
    background-color: rgba(59, 130, 246, 0.3);  /* sirf background */
    /* text poora visible rahega */
}

7. overflow — Content bahar jaaye to kya karein

Jab content apne container se bada ho jaaye — tab kya hoga? overflow se yeh control karte hain.

visible (default) — bahar dikhega
Yeh text container se bahar jaayega kyunki overflow:visible hai by default.
hidden — kaat do
Yeh text container se bahar jaayega lekin overflow:hidden se kaat diya jaayega.
auto — zaroorat par scroll
Overflow auto hai — content zyada hoga to scrollbar automatically aayega. Yeh bahut useful hai.
Common use — long text truncate karo (...)
Yeh bahut lamba text hai jo ek line mein fit nahi hoga...
div { overflow: visible; }  /* default — bahar dikhega */
div { overflow: hidden; }   /* bahar wala cut ho jaayega */
div { overflow: scroll; }   /* hamesha scrollbar */
div { overflow: auto; }     /* zaroorat par scrollbar */

/* Long text truncate — 3 properties saath */
.truncate {
    width: 200px;
    white-space: nowrap;         /* text wrap na kare ek line mein rahe */
    overflow: hidden;            /* bahar kaat do */
    text-overflow: ellipsis;     /* ... dikhao end mein */
}

/* overflow: hidden ka aur ek use */
.card {
    border-radius: 12px;
    overflow: hidden;  /* image card ke border ke bahar nahi jaayegi */
}

8. backdrop-filter — Glass (blur) effect

backdrop-filter element ke peeche wali cheez par filter lagata hai. Sabse common use hai blur — frosted glass jaisa effect.
⚠️ Zaroori: element ka background semi-transparent hona chahiye (rgba) — warna peeche ka blur dikhega hi nahi.

Colored background par glass effect dekhne ke liye

Glass Card

backdrop-filter: blur(10px)

Glass Navbar Style
blur(14px) + rgba bg 
/* Glass card effect */
.glass-card {
    background-color: rgba(255, 255, 255, 0.1);  /* semi-transparent */
    backdrop-filter: blur(10px);                  /* peeche blur */
    border: 1px solid rgba(255, 255, 255, 0.2);
    border-radius: 16px;
    padding: 24px;
}

/* Glass navbar — portfolio mein */
.navbar {
    background-color: rgba(15, 23, 42, 0.85);   /* dark + transparent */
    backdrop-filter: blur(14px);
    position: sticky;
    top: 0;
}

/* Blur amounts */
backdrop-filter: blur(4px);    /* thoda blur */
backdrop-filter: blur(10px);   /* medium */
backdrop-filter: blur(20px);   /* zyada blur */

⚠️ backdrop-filter sirf tab dikhega jab element ke peeche kuch ho — colored/image background hona zaroori hai. Plain white page par effect nazar nahi aayega.

9. background-clip: text — Gradient Text Effect

Text ke andar gradient color dikhane ke liye — portfolio headings, hero titles mein common.
3 steps: (1) gradient set karo, (2) background-clip: text, (3) color: transparent

Blue to Purple Gradient Text
Orange to Red Gradient Text
Rainbow Gradient Text ✨
/* 3 steps zaroori hain */
h1 {
    /* Step 1 — gradient set karo */
    background: linear-gradient(to right, #3b82f6, #8b5cf6);

    /* Step 2 — gradient sirf text ke andar clip karo */
    background-clip: text;
    -webkit-background-clip: text;        /* Safari ke liye */

    /* Step 3 — actual text color transparent karo */
    color: transparent;
    -webkit-text-fill-color: transparent; /* Safari ke liye */
}

/* Agar color: transparent nahi likhoge to
   gradient chhup jaayega — text ka color upar se dikhega */
