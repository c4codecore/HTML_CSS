↔️ CSS Flexbox

Flexbox ek 1D layout system hai — items ko row ya column mein arrange karta hai. Properties do tarah ki hain: Container par lagate hain, ya Item par.

Flexbox Shuru Kaise Karte Hain?

Sirf parent (container) par display: flex likhna kaafi hai. Iske baad container ke andar ke sab direct children automatically flex items ban jaate hain aur ek row mein aa jaate hain. 

/* Container par yeh likhna kaafi hai shuru karne ke liye */
.parent {
    display: flex;
}
/* Ab .parent ke andar ke sab direct children ek row mein aa jayenge */

CONTAINER PROPERTIES — Parent par lagate hain

Yeh properties container (parent) par lagate hain. Yeh andar ke sab items ko affect karti hain.

Items kis direction mein jaayenge — row (horizontal) ya column (vertical)? Default = row.

Jab items container se zyada ho jaayein to kya karein? Default mein items ek hi line mein rehte hain aur overflow ho jaate hain. wrap se neeche wali line mein chale jaate hain.

ITEM PROPERTIES — Individual child par lagate hain

Yeh properties sirf ek specific item par lagate hain — baaki items affect nahi hote.

6. order — Visual order change karo (HTML badlaye bina)

HTML mein jo order likha hai woh change kiye bina CSS se visual order badal sakte hain. Default order = 0. Chhota number pehle aata hai.

HTML order: 1, 2, 3 — lekin CSS order: 3, 1, 2 — display mein Green pehle dikhega
1 (order:3)
2 (order:1)
3 (order:2)
/* HTML mein yeh order hai: 1, 2, 3 */
/* CSS se visual order badal diya */

.item:nth-child(1) { order: 3; }   /* pehla item teesre position par dikhega */
.item:nth-child(2) { order: 1; }   /* doosra pehle dikhega */
.item:nth-child(3) { order: 2; }   /* teesra doosre position par */

7. flex-grow — Bacha hua space kitna lo

Jab container mein extra space ho to kaun kitna lega? Yeh shares ki tarah kaam karta hai. Default = 0 (koi extra space nahi lega — items apne size ke hisaab se).

flex-grow: 0 (default) — items apne content size mein rehte hain
A (0)
B (0)
C (0)
grow: 1, 2, 1 — B ko double space milega kyunki uske 2 shares hain
A (grow:1)
B (grow:2)
C (grow:1)
/* grow: 0 — default, extra space nahi lega */
.item-a { flex-grow: 1; }   /* 1 share */
.item-b { flex-grow: 2; }   /* 2 shares — double space milega */
.item-c { flex-grow: 1; }   /* 1 share */

/* Total shares = 4. Extra space 4 parts mein banta hai.
   A = 1 part, B = 2 parts, C = 1 part */

8. flex-shrink — Container chhota ho to kitna sikudo

Jab container chhota ho jaaye to items shrink hone lagte hain. flex-shrink: 0 se item shrink hona band kar dete hain.
Use case: Logo ya icon jo chhota nahi hona chahiye.

flex-shrink: 1 (default) — sab barabar shrink karte hain

9. flex-basis — Item ki starting size

Item ki initial size set karta hai (flex-grow/shrink se pehle). Flex mein width ki jagah flex-basis use karna better practice hai.

basis: 200px
basis: 100px
basis: auto
.item-a { flex-basis: 200px; }   /* 200px se shuru karega */
.item-b { flex-basis: 100px; }   /* 100px se shuru karega */
.item-c { flex-basis: auto; }    /* content ke hisaab se (default) */

10. flex (shorthand) — grow + shrink + basis ek saath

flex: grow shrink basis — teeno ek jagah likh sakte hain. Sabse common hai flex: 1 — iska matlab hai barabar space lo.

flex: 1
flex: 2 (double)
flex: 1
/* flex: grow shrink basis */
.item { flex: 1; }        /* = flex: 1 1 0% — barabar space lo */
.item { flex: 2; }        /* double space lega compared to flex:1 */
.item { flex: none; }     /* = flex: 0 0 auto — fixed size */
.item { flex: auto; }     /* = flex: 1 1 auto */

11. align-self — Ek item ki khud ki alignment

Container ke align-items ko ek specific item ke liye override karta hai. Baaki items pe koi asar nahi.

Container: align-items: flex-start — lekin individual items alag position mein
flex-start (same as container)
align-self: flex-end
align-self: center
align-self: stretch
.container { align-items: flex-start; }   /* sab upar */

/* Individual item ko override karo */
.item-b { align-self: flex-end; }    /* sirf yeh neeche */
.item-c { align-self: center; }      /* sirf yeh center mein */
.item-d { align-self: stretch; }     /* sirf yeh puri height */
