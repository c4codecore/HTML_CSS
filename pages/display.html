🖥️ CSS Display Property

Har HTML element ka ek default display hota hai. Hum CSS se isko badal sakte hain. Har example mein pehle DEFAULT dikhaya hai, phir BADLAV ke baad kya hota hai.

HTML Elements ke Default Display Values

Yeh values yaad rakho — isi wajah se BEFORE/AFTER examples mein farq dikhta hai.

ElementDefault DisplayMatlab
div, p, h1-h6, ul, sectionblockPuri width, neeche jaata hai
span, a, strong, eminlineSirf content ki width, side-by-side
img, input, buttoninline-blockSide-by-side + size set hoti hai

1. display: block

Block element puri width leta hai aur agla element neeche jaata hai. Width, height, margin, padding sab kaam karte hain.
⚡ Trick: Hum <span> (jo inline hai) pe block lagayenge — tabhi impact dikhega!

❌ PEHLE — span ka default (inline): teeno ek hi line mein
Span 1 Span 2 Span 3
✅ BAAD MEIN — span ko display:block diya: har span apni puri row le raha hai
Span 1 — ab block hai, puri width le raha hai Span 2 — neeche aa gaya Span 3 — width bhi set ho gayi 
/* PEHLE — span ka default inline hai */
span { /* kuch nahi — browser inline maan leta hai */ }

/* BAAD MEIN — block bana diya */
span {
    display: block;    /* ← yeh ek line se sab badal gaya */
    width: 280px;      /* ab width set hogi */
    margin-bottom: 6px;
}

2. display: inline

Inline element sirf content jitni jagah leta hai aur side-by-side rehta hai. Width aur height set nahi hoti — yeh inline ki sabse badi limitation hai.
⚡ Trick: Hum <div> (jo block hai) pe inline lagayenge — tabhi farq dikhega!

❌ PEHLE — div ka default (block): ek ke neeche ek stack ho rahe hain
Div 1
Div 2
Div 3
✅ BAAD MEIN — div ko display:inline diya: teeno side-by-side aa gaye
Div 1
Div 2
Div 3

⚠️ Note: Inline pe width set karo to bhi kaam NAHI karega. Yeh inline ki limitation hai. 

/* PEHLE — div ka default block hai */
div { /* kuch nahi — browser block maan leta hai */ }

/* BAAD MEIN — inline bana diya */
div {
    display: inline;    /* ← side-by-side aa gaye */
    padding: 8px 14px;
    /* width: 200px; — yeh likhne par bhi KAAM NAHI KAREGA */
}

3. display: inline-block

inline-block = inline + block ka combination. Side-by-side bhi rehta hai aur width/height bhi set hoti hai.
Real use case: Navigation links, buttons, gallery items.

❌ PEHLE — span par width/height set ki, lekin kaam nahi ki (inline default)
Box A (120×60 di, ignore hui) Box B Box C
✅ BAAD MEIN — inline-block lagaya: size bhi lagi, side-by-side bhi hain
Box A Box B Box C 
/* PEHLE — span inline hai, width/height kaam nahi karti */
span {
    width: 120px;   /* ignored */
    height: 60px;   /* ignored */
}

/* BAAD MEIN — inline-block se dono kaam hote hain */
span {
    display: inline-block;   /* ← best of both worlds */
    width: 120px;            /* ✅ ab kaam karega */
    height: 60px;            /* ✅ ab kaam karega */
}

4. display: none — vs — visibility: hidden

Dono element ko chhupate hain. Lekin ek zaroori farq hai — layout mein jagah ka!

display: none — Box 2 poori tarah gayab, jagah bhi nahi bachi
Box 1 ✅
Box 2 — display:none
Box 3 ✅

👆 Box 1 aur Box 3 directly adjacent hain — Box 2 ki jagah kisi ne nahi li.

visibility: hidden — Box 2 dikhta nahi, lekin jagah reserve hai
Box 1 ✅
Box 2 — visibility:hidden
Box 3 ✅

👆 Box 2 nazar nahi aata, lekin uski jagah bachi hai — isliye Box 3 door hai. 

/* Poora hata do — jagah bhi nahi lega */
.gone      { display: none; }

/* Sirf chhupao — jagah reserve rahegi */
.invisible { visibility: hidden; }

/* Kab kya use karein?
   display:none      → menu/dropdown toggle, tabs
   visibility:hidden → animation ke liye jagah bachani ho */

5. display: flex

Flex container ke andar items automatically ek row mein aa jaate hain. Alignment, gap, sizing — sab bahut aasaan ho jaata hai.
→ Flex ka poora detail: flex.html mein dekho.

❌ PEHLE — bina flex ke: divs block hain, ek ke neeche ek stack ho rahe hain
Item 1
Item 2
Item 3
✅ BAAD MEIN — display:flex lagaya: sab ek row mein, center aligned
Item 1
Item 2
Item 3
/* PEHLE — default block stacking */
.container { /* kuch nahi */ }

/* BAAD MEIN — flex */
.container {
    display: flex;         /* ← items row mein aa gaye */
    gap: 10px;             /* items ke beech space */
    align-items: center;   /* vertically center */
}

6. display: grid

Grid se rows aur columns dono control kar sakte hain — 2D layout.

/* PEHLE — default block, ek column */
.container { /* kuch nahi */ }

/* BAAD MEIN — grid */
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3 equal columns */
    gap: 10px;
}

📋 Summary — Display Values

Value Puri width? Width/Height? Side-by-side? Use case
block ✅ Haan ✅ Haan ❌ Nahi Headings, paragraphs, sections
inline ❌ Nahi ❌ Nahi ✅ Haan Text ke andar links, bold, span
inline-block ❌ Nahi ✅ Haan ✅ Haan Nav links, buttons, cards ek row mein
none Poori tarah gayab — layout mein jagah bhi nahi Hide/show toggle
flex Children row mein, alignment aasaan Navbar, card row, centering
grid Children rows+columns dono mein Page layout, image gallery
