⊞ CSS Grid Layout

Grid ek 2D layout system hai — rows aur columns dono ek saath. Page layouts, galleries, dashboards — sab Grid se banate hain.

Flexbox vs Grid — Kab kya use karein?

FlexboxGrid
Dimension1D — row ya column2D — rows + columns
DirectionEk direction mein itemsDono directions control
Best forNavbar, button group, card rowPage layout, gallery, dashboard
Shurudisplay: flexdisplay: grid

/* Fixed pixels */
.grid { grid-template-columns: 100px 200px 100px; }

/* fr unit — available space ke proportional */
.grid { grid-template-columns: 1fr 1fr 1fr; }    /* 3 equal */
.grid { grid-template-columns: 1fr 2fr 1fr; }    /* beech wala double */

/* repeat() shorthand */
.grid { grid-template-columns: repeat(3, 1fr); } /* = 1fr 1fr 1fr */
.grid { grid-template-columns: repeat(4, 1fr); } /* 4 equal columns */

/* Mixed */
.grid { grid-template-columns: 150px 1fr 1fr; }  /* sidebar + content */

2. grid-template-rows — Row heights define karo

Default mein rows content ke hisaab se auto size hoti hain. grid-template-rows se specific height de sakte hain.

grid-template-rows: 80px 120px — pehli row 80px, doosri 120px

4. grid-template-areas — Named layout

Cells ko naam deke layout banana — sabse readable aur maintainable tarika. Same naam likhne se woh cells merge ho jaati hain. "." ka matlab empty cell.

HEADER
SIDEBAR
CONTENT
FOOTER
.grid {
    grid-template-columns: 200px 1fr;
    grid-template-rows: 60px 1fr 50px;
    grid-template-areas:
        "header  header"    /* dono columns mein header */
        "sidebar content"   /* pehle column mein sidebar, doosre mein content */
        "footer  footer";   /* dono columns mein footer */
    gap: 10px;
}

/* Har item ko uske naam se assign karo */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer  { grid-area: footer; }

5. auto-fit + minmax() — Responsive Grid (bina media queries)

Yeh combination bahut powerful hai — screen size ke hisaab se columns automatically adjust ho jaate hain.
minmax(180px, 1fr) = column kam se kam 180px, zyada se zyada 1fr.
Browser window chhoti-badi karo aur dekho columns badlenge.

Card 1
Card 2
Card 3
Card 4
Card 5
Card 6
/* Responsive grid — bina media query ke! */
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
    gap: 14px;
}

/* Kaise kaam karta hai:
   - auto-fit  → jitne columns fit hoon utne banao
   - minmax(180px, 1fr) → min 180px, max 1fr
   
   Screen wide ho → 4-5 columns
   Screen narrow ho → 2-3 ya 1 column
   Automatically adjust! */

6. grid-column / grid-row — Item ko multiple cells mein phailao

/* Span — kitne cells cross karo */
.wide-item { grid-column: span 2; }    /* 2 columns lega */
.tall-item { grid-row: span 2; }       /* 2 rows lega */

/* Line numbers se */
.header    { grid-column: 1 / 3; }    /* line 1 se line 3 tak */
.full      { grid-column: 1 / -1; }   /* first se last line tak = full width */

7. justify-items & align-items — Cells ke andar alignment

Grid mein justify-items horizontal aur align-items vertical alignment set karta hai — har cell ke andar. Default = stretch (poori cell bhar lena).

.grid {
    justify-items: stretch;  /* default — cell ki puri width */
    justify-items: start;    /* cell ke left mein */
    justify-items: center;   /* cell ke center mein */
    justify-items: end;      /* cell ke right mein */
    
    align-items: stretch;    /* default — cell ki puri height */
    align-items: center;     /* vertically center */
}

/* Ek item ke liye override */
.item { justify-self: center; }
.item { align-self: end; }

✅ Real World Example — Blog Layout

Ek typical blog page ka layout — grid-template-areas se clearly define kiya.

Hero / Banner
grid-area: hero — Full width
📄 Article
grid-area: article — Main content
📌 Sidebar
grid-area: sidebar — 300px fixed
Footer — Full width
.page {
    display: grid;
    grid-template-columns: 1fr 300px;       /* content aur sidebar */
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "hero     hero"
        "article  sidebar"
        "footer   footer";
    gap: 16px;
}

.hero    { grid-area: hero; }
.article { grid-area: article; }
.sidebar { grid-area: sidebar; }
.footer  { grid-area: footer; }
